# Champion's Forge v8.6 — ETERNAL

**A single-file HTML habit tracker for warriors who build empires.**

No npm. No servers. No tracking.  
All data stored in your browser. Works offline. Forever.

## Live Demo
[championsforge.org](https://championsforge.org)

## Features
- **Smoke Slayer** – Track time since last cigarette  
- **Feast Tamer** – Log meals with notes  
- **Pomodoro Pulverizer** – 25/5, 50/10, custom  
- **Custom Timer / Stopwatch**  
- **Victory Vault** – Log daily wins  
- **Full History + Delete**  
- **Backup/Restore JSON**  
- **Web Audio victory sounds**
